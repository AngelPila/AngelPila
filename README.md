## 💻 Proyectos destacados

### 🎙️ [Govly — Asistente de IA con Reconocimiento de Voz](https://github.com/AngelPila/innovailab)
Asistente de inteligencia artificial para trámites ciudadanos con reconocimiento de voz en tiempo real configurado para español de Ecuador (es-EC). 🥉 3er lugar nacional en el Hackathon Innovailab.
- **Tecnologías**: JavaScript, Web Speech API, React

### 🌱 [WAQI — Plataforma Agritech](https://github.com/AngelPila/WAQI)
Cofundador y desarrollador. Plataforma digital para la transformación del sector agropecuario en Ecuador, conectando directamente a productores, compradores e inversionistas. 🏅 Ganadora del Hackathon Agrofuturo (Nestlé & ESPOL), con potencial de escalabilidad en el Plan Cacao.

### 🛡️ Argly — Detección de Fraude con IA (Insurtech)
Plataforma de análisis predictivo de reportes de siniestros para aseguradoras.
- Diseñé la arquitectura tecnológica y el pipeline de datos de extremo a extremo.
- Modelos de analítica avanzada para calificar el nivel de riesgo de cada reporte, reduciendo los tiempos de auditoría manual.

### 💰 VisuWallet — Gestión Financiera Inteligente
Visualización clara y dinámica de datos financieros personales y empresariales, con componentes interactivos para el seguimiento de activos y UI optimizada para la toma de decisiones.

### 🌍 [Terra Watch Live](https://github.com/AngelPila/terra-watch-live)
Monitoreo ambiental en tiempo real con sistema de diseño personalizado de temas ecológicos. Base del proyecto ganador del NASA Space Apps Challenge local.
- **Tecnologías**: TypeScript, React, Tailwind CSS, shadcn/ui

### 💣 [PyHunt by GrupAP — Bomb Defusal Game](https://github.com/AngelPila/pyhuntbygrupap)
Juego interactivo de desactivación de bombas basado en retos de análisis de datos con Pandas. [🕹️ Jugar en vivo](https://angelpila.github.io/pyhuntbygrupap/)
- **Tecnologías**: JavaScript, CSS, HTML
